# Installation Guide

This guide provides detailed instructions for installing and configuring Artifact Chat in different environments.

## System Requirements

- **Node.js**: v16.x or higher
- **NPM**: v7+ or Yarn v1.22+
- **Memory**: Minimum 2GB RAM for development
- **Storage**: At least 1GB free disk space
- **Operating System**: macOS, Windows 10/11, or Linux

## Standard Installation

<Tabs>
  <Tab title="npm">
    ```bash
    git clone https://github.com/yourusername/artifact-chat.git
    cd artifact-chat
    ```

  </Tab>

## Docker Installation

Artifact Chat can also be run in a Docker container:

```bash
# Clone the repository
git clone https://github.com/yourusername/artifact-chat.git
cd artifact-chat

# Copy and configure environment variables
cp .env.example .env
# Edit .env with your settings

# Start with Docker Compose
docker-compose up -d
```

<Note>
  The Docker setup includes all dependencies and creates an isolated environment
  for the application.
</Note>

## Environment Configuration

Artifact Chat requires several environment variables to be configured:

```bash
# Required API Keys
ANTHROPIC_API_KEY=your_anthropic_api_key
OPENAI_API_KEY=your_openai_api_key

# Optional configurations
VITE_APP_NAME="Artifact Chat"
VITE_APP_DESCRIPTION="AI Development Assistant Platform"
VITE_DEFAULT_MODEL="claude-3-5-sonnet"

# Database Configuration (if using Supabase)
SUPABASE_URL=your_supabase_url
SUPABASE_ANON_KEY=your_supabase_anon_key
```

## Production Deployment

For production deployments, we recommend:

1. **Build the Application**:

```bash
npm run build
# or
yarn build
```

2. **Serve the Production Build**:

```bash
npm run preview
# or
yarn preview
```

3. **Alternative Hosting Options**:

- **Netlify**: Connect your GitHub repository to Netlify for automatic deployments
- **Vercel**: Deploy directly from GitHub with the Vercel platform
- **Docker**: Use the production Docker configuration with `docker-compose.yml`

## Troubleshooting

### Common Issues

1. **Node Version Mismatch**:

   - Error: `Error: The engine "node" is incompatible with this module`
   - Solution: Update Node.js to v16.x or use nvm to switch versions

2. **API Key Issues**:

   - Error: `API key not configured`
   - Solution: Check that your `.env` file includes all required API keys

3. **Build Failures**:
   - Error: `Failed to compile`
   - Solution: Check for dependency issues with `npm clean-install` or clear the cache with `npm run clean:vite`
