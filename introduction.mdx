# Quickstart Guide

This guide will help you get Artifact Chat up and running quickly.

## Prerequisites

Before you begin, make sure you have:

1. **Node.js** (v16 or higher)
2. **npm**, **yarn**, or **pnpm**
3. **API keys** for Anthropic, OpenAI, or other supported LLM providers

## Step 1: Installation

<CodeGroup>
```bash npm
git clone https://github.com/yourusername/artifact-chat.git
cd artifact-chat
npm install
```

```bash yarn
git clone https://github.com/yourusername/artifact-chat.git
cd artifact-chat
yarn install
```

```bash pnpm
git clone https://github.com/yourusername/artifact-chat.git
cd artifact-chat
pnpm install
```

</CodeGroup>

## Step 2: Configuration

Create a `.env` file in the root directory:

```bash
cp .env.example .env
```

Edit the `.env` file to add your API keys:

```
ANTHROPIC_API_KEY=your_api_key_here
OPENAI_API_KEY=your_api_key_here
```

## Step 3: Start the Development Server

<CodeGroup>
```bash npm
npm run dev
```

```bash yarn
yarn dev
```

```bash pnpm
pnpm dev
```

</CodeGroup>

The development server will start at [http://localhost:3000](http://localhost:3000).

## Step 4: Using Artifact Chat

1. Open your browser and go to [http://localhost:3000](http://localhost:3000)
2. Create a new chat or project
3. Start asking questions or providing instructions
